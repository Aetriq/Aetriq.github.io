Chimera iXY

Custom CoreXY 3D Printer
An extremely fast 3D Printer using CoreXY kinematics, built from two different bedslingers. Demonstrating the usability of RepRap in modern printers with professional-grade performance.
Still printing. Still free. Still RepRap.
View on GitHub Full Documentation

Current Status: v1.0 Released! Check out the repository for more details. Work has started on v1.5.

Technical Specifications

Print Speed
200mm/s+

Minimum required printhead speed with quality results.

Build Volume
230mm³

Cubic build surface area. 110C bed with magnetic PEI or glass surface.

Power Consumption
24V 15A - 350W
Hotend
V6

4010x2 part cooling fans

30mm primary hotend cooling fan

Support for all-metal & mk8 (future version)

275C Nozzle

Controller
MKS Robin Nano v3.1

Running Marlin with LVGL UI. Future support for Klipper.

Kinematics
CoreXY
Cost
<100$ (excl. existing printers)
