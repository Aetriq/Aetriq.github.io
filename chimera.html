Chimera iXY

Custom CoreXY 3D Printer
A fast 3D Printer using CoreXY kinematics, built from two different bedslingers; the Flsun i3 2017 and Mingda Magician X. Demonstrating the usability of RepRap in modern printers.
Still printing. Still free. Still RepRap.
View on GitHub Full Documentation

Current Status: v1.5 Released! Check out the repository for more details. v2.0 IS IN THE MAKING!!!

Project Timeline

1-12-2026

RELEASE - v2.0

Second official revision release. Improvements to both software and hardware.

Issues encountered in v1.5 fixed (refer to documentation for all resolved issues)

"Upgraded" to Klipper firmware for addition of input shaping and vibration compensation, which is crucial for speed/quality prints.

More improvements to kinematics and extrusion system to match higher print speeds.

10-18-25

RELEASE - v1.5

First official 'bugfix' of the project finally released.

Includes fixed 3d models to improve speed and print quality.

Marlin firmware addition of MKS WIFI ESP8266 v1.0 Module and BeePrint Wifi interface.

9-15-25

RELEASE - v1.0

First official revision of the project finally released. Documentation and design files uploaded on same day.

Refer to Github for full firmware/CAD Design Files.

9-14-25

TESTING (v0.7-0.9)

Iterative design was done to fix the majority of parts. At this point in time, most frame parts are complete.

7-6-25

ASSEMBLY

General assembly of components complete.

5-24-25

DISASSEMBLY

Disassembly of existing printers complete.

MATERIALS + DESIGNING
